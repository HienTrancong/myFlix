Documentation

Server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

API

Business logic URL HTTP Method Request body data format Reponse body data format
Get a list of all movies /movies GET None JSON object holding data about all movies
Get data about a movie by title /movies/[title] GET None JSON object holding data about a movie.
Example:
 {
 title: 'Slalom',
 rank: 4,
 director: 'Charlène Favier',
 genre: ['Drama','Narrative']
 }
Get data about a genre by genreName /genres/[genreName] GET None JSON object holding data about a genre
Get data about a director by directorname /directors/[directorName] GET None JSON object holding data about a director
Add new user /users POST JSON object holding data about the user to add JSON object holding data about the new user
Update user info (userName) /users/[userName] PUT JSON object holding data about information need to be updated JSON object holding data about the user
Add a movie to user's list of favorites /favourites/[userName]/[movieName] POST None JSON object about updated list of user's favorite movies
Remove a movie from a user's list of favorites /favourites/[userName]/[movieName] DELETE None JSON object about updated list of user's favorite movies
Remove a user by userName /users/[userName] DELETE None A text message indicate the user with userName has been deleted
