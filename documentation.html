Documentation

Server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Add new user /users POST Expect JSON object holding data about the user to add in this format:
{
"Username": String,
"Password": String,
"Email": String,
"Birthday": Date
}

Example:
{
"Username": "test",
"Password": "testpass",
"Email": "test@hotmail.com",
"Birthday": "2000-05-01"
}

Update user info (userName) /users/[userName] PUT Expect JSON object holding data about the user to add in this format:
{

"Email": String, (required)
